# Loja Automotiva – Auto Prime Custom (AC1)

Projeto desenvolvido para a entrega da **AC1**, utilizando **Power BI**, com foco em análise de dados.

## Objetivo
Criar um dashboard interativo para análise da frota de veículos da RMSP e um cenário de faturamento fictício por município, com visão mensal.

## Dados Utilizados
- **Frota de veículos**: base utilizada no projeto, com recorte da RMSP.
- **Faturamento**: dados fictícios, criados exclusivamente para fins acadêmicos.
- Devido ao tamanho do arquivo original de dados de frota, o CSV completo não foi incluído no repositório, pois excede o limite de armazenamento do GitHub. A origem dos dados é pública (RENAVAM) e a análise completa pode ser verificada no arquivo PBIX disponível neste repositório e no vídeo de demonstração da AC1.

## Funcionalidades do Dashboard (AC1)
- Gráfico de Frota por Município
- Gráfico de Faturamento Fictício por Município
- Card de Total de Faturamento
- Segmentador por Mês/Ano
- Dashboard com identidade visual própria
