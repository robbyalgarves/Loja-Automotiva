# Loja Automotiva – Auto Prime Custom (C1)

Projeto desenvolvido para a entrega da **C1**, utilizando **Power BI**, com foco em análise de dados.

## Objetivo
Criar um dashboard interativo para análise da frota de veículos da RMSP e um cenário de faturamento fictício por município, com visão mensal.

## Dados Utilizados
- **Frota de veículos**: base utilizada no projeto, com recorte da RMSP.
- **Faturamento**: dados fictícios, criados exclusivamente para fins acadêmicos.

## Funcionalidades do Dashboard (C1)
- Gráfico de Frota por Município
- Gráfico de Faturamento Fictício por Município
- Card de Total de Faturamento
- Segmentador por Mês/Ano
- Dashboard com identidade visual própria

## Gerenciamento do Projeto
O acompanhamento das atividades foi realizado por meio de um **Board Kanban no GitHub Projects**, com as tarefas da C1 concluídas.
